# Python Hangman Game 🎮
Un classico gioco dell'Impiccato sviluppato in Python, con interfaccia grafica ASCII e gestione dinamica del punteggio.

## ✨ Caratteristiche
- **Game Loop Persistente**: Gioca più partite e accumula il tuo punteggio.
- **Interfaccia Pulita**: Utilizza il modulo `os` per pulire la console a ogni turno.
- **Logica Anti-Errore**: Il sistema riconosce se hai già inserito una lettera (sia corretta che errata) senza penalizzarti.
- **Docstrings & Typing**: Codice scritto seguendo le best practices di Python.

## 🛠️ Tecnologie utilizzate
- Python 3
- Moduli: `random`, `os`, `time`
